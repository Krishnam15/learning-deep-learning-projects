# Deep Learning Projects

## Overview
This repository contains a collection of deep learning projects implemented using Artificial Neural Networks (ANNs).  
Each notebook focuses on applying neural network concepts to real-world and benchmark datasets, with an emphasis on understanding model architecture, training, and evaluation.

## Projects Included

### 1. Bank Customer Churn Prediction
A binary classification problem aimed at predicting whether a customer will churn based on demographic and behavioral features.
- Concepts: Binary classification, ANN modeling, feature preprocessing
- Techniques: Data preprocessing, train-test split, model evaluation

### 2. Graduate Admission Prediction using ANN
A regression-based deep learning project that predicts the probability of graduate admission based on academic and profile-related parameters.
- Concepts: Regression with neural networks
- Techniques: Feature scaling, ANN regression, performance evaluation

### 3. MNIST Classification using ANN
A multi-class classification task on the MNIST handwritten digits dataset using a fully connected neural network.
- Concepts: Multi-class classification, neural network fundamentals
- Dataset: MNIST
- Techniques: One-hot encoding, model training, and evaluation
